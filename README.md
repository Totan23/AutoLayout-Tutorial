# AutoLayout Tutorial

Esta aplicación iOS es un tutorial interactivo de AutoLayout que demuestra cómo crear una experiencia de onboarding con páginas deslizables. La app presenta una colección de pantallas que el usuario puede navegar mediante gestos de swipe o botones de navegación, mostrando imágenes ilustrativas y texto descriptivo. Utiliza UICollectionView con diseño horizontal y controles de página para crear una interfaz fluida y moderna.
